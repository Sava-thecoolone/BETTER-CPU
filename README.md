# BETTER-CPU
A much better CPU i made in Digital Logic Sim Community Edition

# How to open in DLS CE
Paste the folder called `MUCH_BETTER_CPU` into `%appdata%\..\LocalLow\SebastianLague\Digital-Logic-Sim\Projects` and open the project in the DLS

# How to use the assembler
Download the assembler from [here](https://github.com/Sava-thecoolone/BETTER-CPU/releases/tag/Latest) and drag the file you want to compile on top of it. Alternatively, you can run `main.exe` with the file as an argumentin a cmd window for debug info
